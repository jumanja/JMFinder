      Fields description for Config Table
      code:Unique Identifier for the row on this table
      name:Name for the config item of this row
      tipo:Type (Link, Path or Text).
      • Link is for Upper link tools like winmerge, notepad, eclipse
      • Path is for configurations like the path and .exe including params to WinMerge two files
      • Text is to customize CRM Texts and similars
      value:Path or Text for the row
      status:Active, Inactive, Retired.
      • Active -> Normal status
      • Inactive -> Row Deactivated, will be ignored
      • Retired -> Same as Inactive, but in addition a Cleaning process will completly remove this row.



      Fields description for FileTypes Table
      code:Unique Identifier for the row on this table
      value:File Extension with or without wildcards (* or ?) for this filetype
      status:Active, Inactive, Retired.
      • Active -> Normal status
      • Inactive -> Row Deactivated, will be ignored
      • Retired -> Same as Inactive, but in addition a Cleaning process will completly remove this row.


      Fields description for Help Table
      code:Unique Identifier for the row on this table
      value:Content value
      status:Active, Inactive, Retired.
      • Active -> Normal status
      • Inactive -> Row Deactivated, will be ignored
      • Retired -> Same as Inactive, but in addition a Cleaning process will completly remove this row.


      Fields description for Project Managers Table
      code:Unique Identifier for the row on this table
      name:Short Name for this Project Manager (it will appear on ToCRM text)
      status:Active, Inactive, Retired.
      • Active -> Normal status
      • Inactive -> Row Deactivated, will be ignored
      • Retired -> Same as Inactive, but in addition a Cleaning process will completly remove this row.


      Fields description for Projects Table
      code:Unique Identifier for the row on this table
      shortname:Very Short Name for this Project Manager (it will appear on upper toolbar)
      country:Two digits code for Country (a flag icon named that way will be shown if found, on upper toolbar)
      pmanager:Code of Project Manager for this Project (it needs to exist on pmanager table)
      ruta:Physical path of the folder on disk for this project
      rutaCRM:Source Safe path for this project (it will appear on ToCRM text)
      orden:Value (usually a number) used as the order this project will appear on toolbar
      status:Active, Inactive, Retired.
      • Active -> Normal status
      • Inactive -> Row Deactivated, will be ignored
      • Retired -> Same as Inactive, but in addition a Cleaning process will completly remove this row.


      Fields description for Tables Table
      code:Unique Identifier for the row on this table
      name:Table Name
      status:Active, Inactive, Retired.
      • Active -> Normal status
      • Inactive -> Row Deactivated, will be ignored
      • Retired -> Same as Inactive, but in addition a Cleaning process will completly remove this row.
